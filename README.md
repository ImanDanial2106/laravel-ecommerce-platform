# Laravel E-Commerce Platform

A robust e-commerce solution built with **Laravel 10**, **PHP 8.4**, and **MySQL**.

## Features
* **Product Catalog:** Browse and filter products.
* **Shopping Cart:** Session-based cart management.
* **Checkout Flow:** Integrated order processing.
* **Admin Panel:** Manage inventory and orders.

## Tech Stack
* **Framework:** Laravel 10
* **Database:** MySQL
* **Frontend:** Tailwind CSS + Blade
* **Dev Environment:** Laravel Herd

## Installation & Database Setup
Since this project uses MySQL, follow these steps to run it locally:

1.  **Clone the repo:**
    ```bash
    git clone [https://github.com/ImanDanial2106/laravel-ecommerce-platform.git](https://github.com/ImanDanial2106/laravel-ecommerce-platform.git)
    cd laravel-ecommerce-platform
    ```

2.  **Install Dependencies:**
    ```bash
    composer install
    npm install && npm run build
    ```
